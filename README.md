# StackTrack

A bullion broker platform for tracking precious metals transactions, inventory, and profitability.

## Features

- **Deal Calculator** — Real-time pricing engine with configurable premiums/discounts for gold and silver (coins, bars, rounds, junk silver, scrap)
- **Live Spot Prices** — Auto-updating gold and silver spot prices via API
- **Transaction Log** — Full transaction history with sorting, filtering, and CSV export
- **Inventory Tracking** — Real-time gold and silver holdings with per-form breakdowns
- **P&L Dashboard** — Daily/weekly/monthly profit and loss with capital position tracking
- **Customer Profiles** — Customer management with transaction linking
- **Wholesale Contacts** — Directory for wholesale dealer contacts
- **Receipt Generation** — Printable transaction receipts
- **Form 8300 Compliance** — Automatic flagging of cash transactions >= $10,000 with review workflow, alerts, and compliance filters
- **Dark/Light Theme** — Toggle between dark and light modes

## Getting Started

Open `index.html` in a browser. No build step or server required — everything runs client-side with localStorage for persistence.

## Tech Stack

Single-file HTML/CSS/JS application with no external dependencies.
